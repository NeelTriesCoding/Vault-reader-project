## INSTALL [ instructions inside ]

```
REQUIRED: Android Studio
```

```bash
1. open Android Studio
2. File > Open > select this folder
3. let it sync, fix whatever it whines about
4. create a .env file in the project root
5. drop your key in:  GEMINI_API_KEY=your_key_here
   (see .env.example if you're lost)
6. in app/build.gradle.kts, delete this line:
   signingConfig = signingConfigs.getByName("debugConfig")
7. hit run. pick an emulator or plug in your phone.
```
